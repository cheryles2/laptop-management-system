# React Laptop Shop UI

Modern laptop storefront built with React + TailwindCSS.

## Features

- Navbar with logo, search, cart, and account
- Sidebar filters: brand, price, RAM, CPU
- Product cards with image, specs, price, and add-to-cart
- Loading skeleton state
- Responsive grid: mobile 1 column, tablet 2 columns, desktop 4 columns
- Mock data with 8 products

## Run

```bash
npm install
npm run dev
```

## Tech

- React
- TailwindCSS
- Vite
- Lucide React
